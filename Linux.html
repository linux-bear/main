Penguins and Bears

My love of Linux

I've been using Linux since 2007, and have been using it as my only system since they made it possible to record music on it in about 2010, when the clever awesome devs made it possible to use 2 soundcards simultanously, so one can listen to the track through headphones while recording in the mic.

I use the Terminal for most administration tasks, I don't think I've ever used the Fedora software management GUI, dnf is a powerful commandline tool and is far preferable.

I used only Linux for my university work, even though at times it has been perplexing shoehorning their software in, sometimes using Wine or finding an alternative and adapting their instructions to my development environment. It provides a learning curve and often a deeper understanding of the subject matter.
