The BearBase

MySQL

There will be a mysql database embedded into the contacts section of this website. It's been a good learning curve, and the code thus far is available at Github although it hasn't been implemented as yet, since the BearBase is a work in progress, and requires security adjustments before it goes live.

The BearBase does work locally, and saves the inputted data to the database table. I run MySQL in Terminal in Fedora 31, and write the code in Visual Studio. The php server is run from Terminal, using the command php -S localhost:8000, then running localhost:8000/myPage.php in Chrome.
